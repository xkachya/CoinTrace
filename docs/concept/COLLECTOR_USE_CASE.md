## 1. Що таке CoinTrace™ простими словами

CoinTrace™ — **кишеньковий прибор** на базі M5Stack Cardputer (ESP32-S3),
який вимірює як метал монети "гальмує" електромагнітне поле на чотирьох
різних відстанях і будує "відбиток" — унікальний профіль кривої загасання.

**Ключова ідея:** кожен метал і кожна проба загасають сигнал по-своєму.
Якщо дві монети мають однаковий відбиток — вони зроблені з одного металу.

### Порівняння з аналогами

| | Сигма/XRF ($1000–5000) | Купелювання (руйнівне) | Магніт/звук | **CoinTrace™ (~$60)** |
|---|---|---|---|---|
| Визначає метал | Так | Так | Частково | **Так** |
| Руйнує монету | Ні | **Так** | Ні | **Ні** |
| Визначає пробу | Точно | Точно | Ні | Приблизно |
| Виявляє підробку | Так | Так | Частково | **Так (більшість)** |
| Ціна | $1000–5000 | — | $0–5 | **~$60** |
| Відкритий код | Ні | — | — | **GPL v3** |

## 2. Що виявляє — і що не виявляє

### ✅ Виявляє надійно

| Ситуація | Приклад |
|---|---|
| Срібло vs сталь | Підробна "срібна" монета з магнітної сталі |
| Срібло vs алюміній | Легкий зліток під срібло |
| Срібло vs вольфрам | Вольфрамовий core (популярна підробка золота/срібла) |
| Срібло vs свинець | Свинцевий core |
| Феромагнітний core | Будь-який залізний/нікелевий/кобальтовий наповнювач |

### ⚠️ Виявляє, але потрібна калібрування

| Ситуація | Точність без еталону | Точність з еталоном |
|---|---|---|
| Ag999 vs Ag925 | Ненадійно | Добре |
| Ag925 vs Ag900 | Ненадійно | Добре |
| Ag900 vs Ag833 | Ненадійно | Задовільно |
| Мідний core без покриття | Задовільно | Добре |

### ❌ Фізично не може виявити

| Ситуація | Причина |
|---|---|
| Мідний core з > 150 µm срібним покриттям | Сигнал проникає лише на 67 µm — "бачить" тільки срібло |
| Проба ±10 (напр. 925 vs 915) | Різниця провідності < 2% — менша за шум сенсора |

> **Важливо:** Навіть XRF не виявить якісно нанесений 150-мікронний шар срібла, якщо не зробити поперечний зріз монети. Це фундаментальне фізичне обмеження, не вада CoinTrace.

## 3. Ти вже маєш ідеальну точку відліку

Якщо в тебе є **офіційні монети НБУ проби 999** — це найкращий з можливих
еталонних зразків:

- **Склад сертифікований** державним монетним двором (MINT Certificate)
- **Проба 999** — найчистіше доступне срібло
- **Задокументовані** в каталогах НБУ (вага, діаметр, рік, тираж)
- **Без сумнівів** щодо автентичності раз назавжди

### Що це означає для CoinTrace

```
NBU Ag999 монета виміряна → записується в базу як "еталон Ag999"
                            ↓
Усі майбутні вимірювання порівнюються з цим еталоном
                            ↓
"Схоже на Ag999 на 94%" — надійна відповідь
"Схоже на мідь на 87%" — підозра на підробку
```

**Одна вимірювана NBU монета прив'язує всю систему** до реального Ag999.
Після цього 500 монет з різних джерел класифікуються відносно неї.

## 4. Перший запуск: 5 монет за 30 хвилин

Рекомендована процедура первинного калібрування:

| Крок | Що виміряти | Навіщо | Де взяти |
|---|---|---|---|
| 1 | **NBU Ag999** (є у тебе) | Еталон чистого срібла | Вже є |
| 2 | Ag925 (будь-яка інша відома) | Еталон срібла 925 | Ювелірний або нумізматичний |
| 3 | Мідна монета | Еталон Cu | СРСР 1-коп / 2-коп / 3-коп (до 1992) |
| 4 | Сталева монета | Еталон Fe | Будь-яка пост-рад. монета що магнітна |
| 5 | Алюмінієва монета | Еталон Al | СРСР 1-коп (1961–1991) |

**Результат:** 5 записів у базі = система знає де "живе" кожен метал в
координатному просторі твого конкретного сенсора. Подальші виміри —
просто пошук найближчого сусіда.

## 5. Польовий сценарій: ярмарок/аукціон

CoinTrace — **портативний** пристрій:
- Живлення від вбудованого акумулятора 1750 мAh (не потрібна розетка)
- Розмір — трохи більший за смартфон
- Дисплей 1.14" з підсвіткою, видно на сонці
- Клавіатура для навігації меню

### Сценарій "нумізматичний ярмарок"

```
Ситуація: Продавець пропонує "срібний крейцер Австрія 1780-х, Ag833".
          Просить $45. Виглядає автентично.

Крок 1: Дістаєш Cardputer з кишені. Вмикається за 2 секунди.

Крок 2: Продавець кладе монету на платформу сенсора.
        CoinTrace виконує 4 виміри (4 × 10 секунд = ~40 секунд).

Крок 3: Екран показує:
        ┌─────────────────────────┐
        │ РЕЗУЛЬТАТ АНАЛІЗУ       │
        │                         │
        │ Метал:   СРІБЛО  ✓      │
        │ Проба:   ~800–850       │
        │                         │
        │ Найближчий еталон:      │
        │   Ag833  92%            │
        │   Ag900  71%            │
        │                         │
        │ Феромагн. core:  НІ ✓   │
        │ Підозра підробки: НІ ✓  │
        └─────────────────────────┘

Крок 4: Приймаєш рішення на основі даних.
```

### Сценарій "виявлена підробка"

```
Ситуація: "Срібна" монета виглядає ідеально. Але:

        ┌─────────────────────────┐
        │ РЕЗУЛЬТАТ АНАЛІЗУ       │
        │                         │
        │ Метал:   СТАЛЬ  ⚠️       │
        │ Проба:   N/A            │
        │                         │
        │ Феромагн. core:  ТАК ❌  │
        │ Підозра підробки: ТАК ❌ │
        │                         │
        │ Тест магнітом: перевір  │
        └─────────────────────────┘

→ Монета будь-якого вигляду, але со стальним core.
  Перевіряєш магнітом — підтверджується за 1 секунду.
```

## 6. 500 монет → особиста база

Твоя колекція 500 монет — це **актив**, а не просто об'єкт дослідження.

### Як формується особиста база

```
Вимір 1:  NBU Ag999 (відомий) → еталон Ag999
Вимір 2:  Ag925 (відомий) → еталон Ag925
...
Вимір 50: Австрійський таллер 1780 → кластер "Ag833 відомих"
Вимір 51: Схожа монета → порівнюється з таллером
          Відстань 0.03 → "дуже схожа" (той самий тип)
          Відстань 0.41 → "слабка схожість" (підозра)
```

### Що можна робити з базою 500 монет

| Можливість | Опис |
|---|---|
| **Ідентифікація дублікатів** | "Ця монета ідентична тій, що в скриньці №3" |
| **Кластеризація по металу** | Автоматично групує Ag925, Ag900, Ag833 |
| **Аномалії** | Монети що "не вписуються" → підозрілі або рідкісні |
| **Еволюція колекції** | Нові монети порівнюються з усієї накопиченою базою |
| **Перевірка покупок** | "Ця купівля відповідає тому що вже є в базі?" |

## 7. Community: колективна база відбитків

### Концепція

Відбитки нормалізовані по розміру монети (доведено математично):
значення **k1** і **k2** у твоєї 20-гривневої НБУ монети і у такої ж
монети твого друга будуть **однаковими**, незалежно від діаметру.

Це дозволяє будувати **спільну базу**:

```
Ти вимірюєш:    Австрійський 1 florin 1858, Ag900 → запис в DB
                {metal:"Ag900", year:1858, mint:"Wien", k1:0.742, k2:0.391}

Інший користувач в Польщі:
                Така сама монета → DB повертає "знайдено 1 збіг, 97%"
                → підтвердження: автентична Ag900
```

### Умови сумісності між пристроями

Щоб відбитки двох різних CoinTrace були порівнянними, потрібно:

| Параметр | Значення | Задається |
|---|---|---|
| `protocol_id` | `p1_1mhz_013mm` (canonical) | Firmware + `protocols/registry.json` |
| Частота сенсора | 1.0 MHz | Прошивка (фіксовано для p1) |
| Відстані вимірювань | 0 мм, 1 мм, 3 мм | Стандартні spacers |
| Модель котушки | LDC1101 Click MIKROE-3240 | BOM проекту |

Якщо `protocol_id` однаковий — відбитки сумісні між будь-якими двома
пристроями CoinTrace у будь-якому місці світу. Нові конфігурації (інша
частота, інші відстані) отримують власний `protocol_id` і не ламають
існуючі записи.

### Вклад в community

```
500 виміряних монет колекційної вартості + публічна ліцензія CC BY-SA →
найбільший внесок одного автора в базу CoinTrace на момент запуску
```

## 8. Підсумок: чого очікувати реально

### Сильні сторони CoinTrace для твоєї задачі

| Задача | Реалістичність |
|---|---|
| Відсіяти сталь/алюміній/вольфрам замість срібла | ✅ Відмінно |
| Підтвердити "це срібло" before buying | ✅ Добре |
| Порядок проби (999 / 925 / 900 / 833) | ✅ З еталонами добре |
| Точна проба ±10 | ❌ Не може (фізика) |
| Виявити феромагнітний core | ✅ Відмінно (через ΔL) |
| Виявити мідний core (без покриття) | ✅ Задовільно |
| Виявити мідний core з товстим сріблом | ❌ Фізично неможливо |
| Ідентифікувати тип монети (emperor/рік/монетний двір) | 🔄 З достатньою базою — можливо |

### Твоя ситуація: оптимальніша за середню

1. **Маєш NBU Ag999** — готовий еталон Ag999, який навіть у 100% готовій системі
   не поліпшити. Кращого старту не існує.

2. **Маєш 500 монет** — не проблема "де взяти зразки", а готова база для
   навчання системи.

3. **Срібна колекція** — найбільш цікавий кейс для CoinTrace, бо срібло
   варіюється від 999 до 500 проби з різними сплавами.

4. **Підробки** — перевірка перед покупкою = основний use case,
   для якого система проектувалась.
