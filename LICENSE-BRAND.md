# CoinTrace™ Trademark and Brand Usage Guidelines

**Trademark Owner:** Yuriy Kachmaryk  
**First Use Date:** 2026-03-08  
**Location:** Lviv, Ukraine  

## Trademark Status

**CoinTrace™** is a trademark claimed by Yuriy Kachmaryk under common law trademark rights. The ™ symbol indicates an unregistered trademark. Formal trademark registration is planned.

## Permitted Use

You MAY use the name "CoinTrace" in the following contexts without explicit permission:

### ✅ Allowed Uses

1. **Factual References**
   - "This device uses CoinTrace™ firmware"
   - "Compatible with CoinTrace™ protocol"
   - "Based on CoinTrace™ open-source design"

2. **Community Contributions**
   - Naming your fork/derivative as "MySoftware for CoinTrace™"
   - Blog posts, tutorials, reviews about CoinTrace™
   - Academic papers citing CoinTrace™ technology

3. **Non-Commercial Projects**
   - Personal builds using CoinTrace™ design
   - Educational demonstrations
   - Research projects

4. **Technical Documentation**
   - Acknowledging use of CoinTrace™ code/design
   - Compatibility statements
   - Technical comparisons

### ✅ Attribution Requirements

When using "CoinTrace" in permitted contexts:

- Use the ™ symbol on first prominent mention: "CoinTrace™"
- Include attribution:  
  ```
  CoinTrace™ is a trademark of Yuriy Kachmaryk.
  ```
- Do not imply endorsement or official affiliation unless authorized

## Prohibited Use

You MAY NOT use "CoinTrace" in the following ways without written permission:

### ❌ Forbidden Uses

1. **Commercial Brand Use**
   - Selling products named "CoinTrace [anything]"
   - Using "CoinTrace" as your company/product name
   - Registering domains like "cointrace-official.com" for commercial purposes

2. **Confusingly Similar Names**
   - "CoinTrace Pro", "CoinTrace Plus", "CoinTrace Elite"
   - "Coin Trace", "CoynTrace", "CoinTracer" (confusingly similar)

3. **Misleading Use**
   - Implying official endorsement: "Official CoinTrace distributor" (without authorization)
   - Suggesting you are the trademark owner
   - Modifying the logo and claiming it as official

4. **Domain Registration**
   - Primary domains: cointrace.com, cointrace.org, cointrace.net are reserved
   - You MAY use: "myproject-for-cointrace.com" (clearly distinct)

The CoinTrace™ logo and visual identity are proprietary. If you wish to use official logos:

- Request permission via: GitHub Issues or direct contact
- For community projects: Modified/derivative logos are encouraged
- Do not use official logo in a way that implies false affiliation

## Open Source ≠ Open Brand

**Important Distinction:**

- ✅ **Code, hardware, docs**: Licensed under GPL v3 / CERN OHL v2 / CC BY-SA 4.0  
  → You can freely use, modify, distribute
  
- ⚠️ **Brand name "CoinTrace™"**: Trademark protection  
  → Limited use as described in this document

This is standard practice. Examples:
- Mozilla Firefox® (open-source code, protected brand)
- WordPress® (GPL code, trademarked name)
- Red Hat® (GPL software, trademarked brand)

The code/hardware remain fully open. The trademark prevents market confusion and protects the project's reputation.

## Fair Use

This policy does not restrict:

- Nominative fair use (referring to CoinTrace™ to discuss the actual project)
- Parody or criticism
- News reporting or commentary
- Academic citations

## Enforcement

We will enforce this policy to:

- Prevent consumer confusion
- Maintain project quality and reputation
- Ensure users know what is "official" vs community-made

Violations may result in:
1. Friendly request to comply with guidelines
2. Formal cease-and-desist if necessary
3. Legal action only as a last resort

Our goal is cooperation, not litigation. Most issues can be resolved with clear communication.
